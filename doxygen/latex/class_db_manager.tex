\doxysubsection{Detailed Description}
S\+QL Database Manager class. 

\mbox{\hyperlink{class_db_manager}{Db\+Manager}} sets up the connection with S\+QL database and performs some basics queries. The class requires existing S\+QL database. You can create it with sqlite\+:

\item \$ sqlite3 people.\+db
\item sqilte$>$ C\+R\+E\+A\+TE T\+A\+B\+LE people(ids integer primary key, name text);
